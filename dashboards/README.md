# Dashboards

This folder contains business intelligence dashboards created using Power BI.

The dashboards consume analytical data from the Gold Layer and provide insights into retail business performance.

## KPIs

The dashboard includes:

- Total Revenue
- Total Orders
- Average Order Value
- Sales Trends
- Top Performing Products
- Customer Segmentation
- Category-wise Sales Analysis

## Dashboard Architecture

Gold Layer
     
     ↓

Snowflake Data Warehouse

     ↓

Power BI Dashboard
